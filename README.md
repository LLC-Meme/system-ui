# Meme System UI  
Memeの社内システムで用いるReactコンポーネントライブラリです。  
Apple風のデザインで様々なUIを提供します。  

## 概要  
- 全てReact/TypeScriptを用いて開発  
- Reactが使えるどのフレームワークでも利用可能  
- Storybookでドキュメント化  
- ChromaticでStorybookをデプロイ  
- デプロイはGitHub Actionsで自動化  

## 依存関係  
- react/react-dom  
- storybook  
- tailwind  
- radix  

## ネクストステップ  
- Form周りのAPIの命名考える、LabelTextには同じの使うとか  
- FormItemのエラー時のUI  
- Theme Switcher作る  
- eslint, prettierを追加  
- huskyの導入  
- ライブラリ化してnpmで使えるようにする  
- ライブラリで問題があれば解決  
- 可能であればstyle定義を呼び出した側にも影響させる  
- ドキュメントを丁寧に書く
