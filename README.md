# Meme System UI  
Memeの社内システムで用いるReactコンポーネントライブラリです。  
Apple風のデザインで様々なUIを提供します。  

## 概要  
- 全てReact/TypeScriptを用いて開発  
- Reactが使えるどのフレームワークでも利用可能  
- Storybookでドキュメント化  
- ChromaticでStorybookをデプロイ  
- デプロイはGitHub Actionsで自動化  

## 依存関係  
- react/react-dom  
- storybook  
- tailwind  
- radix  

## ネクストステップ  
- style定義を呼び出した側にも影響させる  
- npm importに死ぬほど時間かかる問題を解決  
- ドキュメントを丁寧に書く
